## Keyboard Interactions:
- ArrowUp/ArrowRight: Increment value
- ArrowDown/ArrowLeft: Decrement value
- Home: Set to minimum value
- End: Set to maximum value
- Shift + Arrow keys: Larger steps
- Tab: Navigate between thumbs (in range mode)

## API:

### Root:
```typescript
interface SliderProps {
  isRange?: boolean;
  value?: number | [number, number] | Signal<number | [number, number]>;
  min?: number;
  max?: number;
  step?: number;
  disabled?: boolean | Signal<boolean>;
  onChange$?: (value: number | [number, number]) => void;
  onChangeEnd$?: (value: number | [number, number]) => void;
}
```

# Headless ScrollArea Component

## API

```
<ScrollArea.Root>
  <ScrollArea.ViewPort>
    {/* data */}
  <ScrollArea.ViewPort/>
  <ScrollArea.ScrollBar> {/* horizontal */}
    <ScrollArea.Thumb />
  <ScrollArea.ScrollBar/>
  <ScrollArea.ScrollBar> {/* vertical */}
    <ScrollArea.Thumb />
  <ScrollArea.ScrollBar/>
</ScrollArea.Root>

```

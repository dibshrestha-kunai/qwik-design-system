# Radio Group Research

## Overview
Radio Group component allows users to select a single option from a list. The component is fully accessible,
supports keyboard navigation, and complies with the WAI-ARIA specification for radio groups.

## Research Links:
- WAI-ARIA Radio Group Pattern (https://www.w3.org/WAI/ARIA/apg/patterns/radio/)
- Ark UI Radio Group (https://ark-ui.com/docs/components/radio-group)
- Kobalte Radio Group (https://kobalte.dev/docs/core/components/radio-group)
- Radix UI Radio Group (https://www.radix-ui.com/primitives/docs/components/radio-group)

## Keyboard Interactions:
- Tab: Focus on first item (if none selected) or selected item
- Space/Enter: Select focused item
- ArrowDown/ArrowRight: Move to next item
- ArrowUp/ArrowLeft: Move to previous item
- Home: Move to first item
- End: Move to last item

# FileUpload

A component that enables users to upload files through drag-and-drop or file selection.

<Showcase name="hero" />

## Basic Usage

The FileUpload component provides an interface for file uploads through drag-and-drop or file selection.

```tsx
<FileUpload.Root>
  <FileUpload.Input />
  <FileUpload.Dropzone>
    <p>Drop files here or</p>
    <FileUpload.Trigger>Browse</FileUpload.Trigger>
  </FileUpload.Dropzone>
</FileUpload.Root>
```
